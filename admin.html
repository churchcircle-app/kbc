NFC Setup
How to program and deploy your circles
1
Order your NFC tags
Get NTAG215 round epoxy stickers from Amazon — durable, waterproof, perfect for chair backs. About $15 for 50 tags.

For chair backs: 30mm round epoxy stickers.
For lobby/café: standard white sticker tags work great.
🛒 Search Amazon Canada →
2
Download NFC Tools app
Free app on iOS and Android. Open it → Write → Add Record → URL. Program each circle type with its URL below. All chair-back circles use the same auditorium URL — you only scan them once ever.
🪑 Auditorium URL (program all chair tags with this)
https://kbc.churchcircle.app/?circle=auditorium
☕ Lobby URL
https://kbc.churchcircle.app/?circle=lobby
🧒 Kids URL
https://kbc.churchcircle.app/?circle=kids
0
Stick & deploy
Place tags centre-bottom of seat backs. Add a small printed label above that says "Tap to connect →" — this alone doubles your tap rate.

For lobby, stick to the café counter edge or on a small acrylic stand.
4
Update links remotely anytime
The tag URL never changes. When you edit a circle here and hit Save, it updates instantly for every person who taps — no touching the physical tags ever again.

Swap links for special events, update your giving campaign, or turn circles off temporarily — all from this dashboard.
